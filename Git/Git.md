+ 说明
windows环境下使用Git和TortoiseGit，协同操作，个人习惯与远程代码库交互使用Git Bash，本地代码库操作使用TortoiseGit

+ 基础命令

```
git pull                    拉取
git push                    提交
git pull origin master      和主分支master代码保持同步
git branch -av              查看存在的分支
git checkout branch_name    切到需要合并的分支
git fetch -p                删除远端已经没有的分支
```
